# Credit Scoring Model

## Project Overview
This project predicts whether a customer is likely to default on a loan based on financial and behavioral features.

## Dataset
- 1000 customer records
- Income
- Savings
- Debt
- Spending categories
- Credit score
- Default status

## Workflow
1. Data Cleaning
2. EDA
3. Feature Engineering
4. Handling Imbalanced Data using SMOTE
5. Model Training
6. Model Comparison
7. Ensemble Model Selection

## Models Used
- Logistic Regression
- Random Forest
- Gradient Boosting
- XGBoost
- Voting Ensemble

## Results
Best Model: Ensemble (Tuned)

Accuracy: 73.5%

ROC-AUC: 0.654
